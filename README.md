```sh
├─input: input datas of assembly project

|   |   └─FURNITURE_NAME

├─output: result generated by going-on-assembly

|   |   |   |       status_4.yaml
|   |   |   |
|   |   |   ├─instruction_2
|   |   |   └─instruction_3
|   |   |   ...
|   |   └─FURNITURE_NAME
|   |
```
